# selenium_news_crawling
### 셀레니움을 이용한 네이버 뉴스 크롤링
코드 진행 순서
1. 네이버 뉴스 **검색어 입력**
2. 크롤링할 **페이지 수 입력**
3. 해당 페이지 로드
4. 페이지별 **언론사, 기사제목, 주소링크**를 크롤링
5. 크롤링한 결과를 **텍스트 파일로 저장**
